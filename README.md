# Sakura-Ramen

🍜 A fresh, responsive Ramen Restaurant website crafted with pure HTML5 & CSS3. Delicious design, zero frameworks.

## 🛠️ Tech Stack
* **HTML5:** Semantic markup for a structured layout.
* **CSS3:** Custom styling, Flexbox/Grid for responsiveness, and smooth transitions.

## ⚜️ Features
* Responsive design (optimized for mobile, tablet, and desktop)
* Clean and modern UI showcasing a ramen shop's menu
* Built entirely from scratch
